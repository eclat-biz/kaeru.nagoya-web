# kaeru.nagoya-web

かえる調剤薬局 WordPress テーマ改修プロジェクト。

## 対象サイト

https://www.kaeru.nagoya/

## 対象テーマ

実際のWordPressテーマフォルダ名は `kaeru`。  
GitHub上でフォルダ名が日本語表示になっている場合でも、作業上は `kaeru` テーマとして扱う。

## 改修目的

1. 現状解析（テーマ構成・問題点の洗い出し）
2. デザインのリニューアル
3. スマホ最適化・表示速度改善
4. 漢方相談・ペット漢方ページの整備
5. note・Threadsへの導線追加
6. AI相談チャットの実装
7. 予約・問い合わせ機能の強化

## 進め方

一度にすべて実装せず、段階的に進める。

### Phase 1: 基盤修正

- jQuery差し替えの廃止
- CSS/JS読み込みを `functions.php` の `wp_enqueue_scripts` に集約
- viewport追加
- `wp_body_open()`追加
- Google Mapsをトップページだけで読み込む
- `common.js`の地図エラー対策
- 未定義変数の修正
- 最低限のエスケープ対応

### Phase 2: スマホ最適化・表示速度改善

- 960px固定幅の解消
- レスポンシブ対応
- 画像軽量化
- Google Maps遅延読み込み
- 不要CSS/JS削減
- フォント読み込み見直し

### Phase 3: デザインリニューアル

- 既存の雰囲気を残しつつ現代的なデザインへ更新
- 高齢者にも読みやすい文字サイズ・余白
- 漢方相談・ペット漢方の専門性を伝える
- スマホで電話・予約しやすい導線を設計

### Phase 4: 漢方相談・ペット漢方ページ整備

- 漢方相談ページ
- ペット漢方ページ
- 初めての方へ
- 相談の流れ
- よくある質問
- 予約・問い合わせ導線

### Phase 5: note・Threads導線追加

- note記事へのリンク
- Threadsプロフィールへのリンク
- 最新情報・読み物への導線

### Phase 6: 予約・問い合わせ機能強化

- 電話ボタン
- LINE相談
- メールフォーム
- 予約フォーム
- 営業時間・注意事項の明確化

### Phase 7: AI相談チャット実装

- 診断・処方判断は行わない
- 個人情報を入力させない
- 一般的な案内と相談導線に限定する
- 最終的には有人相談へ誘導する

## 注意事項

- 本番環境へ直接反映しない
- まずステージング環境で確認する
- 変更前に必ずバックアップを取得する
- 1フェーズずつ実装・確認する
- 医療・薬機法・個人情報に関わる表現には注意する
