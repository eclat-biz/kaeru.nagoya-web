# kaeru-renewal

かえる調剤薬局の新サイト制作用テーマです。

この初期PRでは、完成ページ制作ではなく、WordPressテーマとして認識・検証できる最小基盤の追加だけを行います。

## 方針

- 既存テーマ `kaeru/` は変更しない
- CSS/JSは `functions.php` の `wp_enqueue_scripts` で読み込む
- `wp_head()` / `wp_footer()` / `wp_body_open()` を最初から使用する
- 出力箇所では `esc_html()` / `esc_url()` / `esc_attr()` を適切に使用する
- スマホ前提で拡張しやすいCSSにする
- LINE予約・問い合わせ導線は今後のフェーズで追加する
